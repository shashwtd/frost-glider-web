# Frost Glider

A 2D infinite side-scrolling snowboarding game. Glide across smooth, procedurally generated hills while avoiding enemies lurking beneath the ice.

**Theme:** Beneath the Surface

## Concept

- Infinite horizontal snowboarding on smooth terrain
- Procedurally generated hills and slopes
- Enemies emerge from beneath the ice
- Score increases the longer you survive
- Inspired by Alto's Adventure

## Status

Work in progress - hackathon project.
